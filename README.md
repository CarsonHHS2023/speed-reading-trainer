快速阅读训练程序
==================

⚡ 一个功能完整的在线快速阅读训练工具，帮助您提升阅读速度和理解力。

## 🎯 主要功能

### 📁 文件支持
- ✅ 支持 **TXT** 文本文件上传
- ✅ 支持 **PDF** 文档解析
- 自动提取和处理文本内容

### 🌍 语言支持
- **中文模式**：按字分割，显示"字/分钟"速度
- **英文模式**：按词分割，显示"词/分钟"速度

### ⚙️ 自定义设置

#### 阅读速度
- 中文范围：100-2000 字/分钟
- 英文范围：100-2000 词/分钟
- 使用滑块或直接输入精确调整

#### 显示格式
- **行宽**：10-50 个字/词
- **行数**：2-10 行

#### 视点移动方式
1. **固定式**（焦点固定）
   - 文字在焦点位置切换
   - 减少眼球移动，提高效率
   - 适合初级训练

2. **滚动式**（页面滚动）
   - 文字缓慢向上滚动
   - 呈现更自然的阅读体验
   - 适合高级训练

### 🎮 控制功能
- **开始阅读**：从第一个单位开始阅读
- **暂停**：暂停当前阅读
- **继续**：从暂停位置继续阅读
- **停止**：停止阅读并返回初始状态

### 📊 进度追踪
- 实时显示当前位置和总词数
- 动态进度条
- 阅读用时统计

## 🚀 快速开始

### 1. 上传文件
点击"选择TXT或PDF文件"按钮，选择要阅读的文件。

### 2. 配置设置
- 选择语言（中文/英文）
- 设置阅读速度
- 调整行宽和行数
- 选择视点移动方式

### 3. 开始训练
- 点击"开始阅读"按钮启动
- 使用"暂停"和"继续"控制节奏
- 点击"停止"结束当前会话

## 📋 使用建议

### 初级训练（推荐设置）
```
速度：300-400 字/词/分钟
行宽：20-25 字/词
行数：5-6 行
模式：固定式
```

### 中级训练（推荐设置）
```
速度：500-700 字/词/分钟
行宽：25-30 字/词
行数：6-8 行
模式：固定式或滚动式
```

### 高级训练（推荐设置）
```
速度：800-1000+ 字/词/分钟
行宽：30-40 字/词
行数：8-10 行
模式：滚动式
```

## 💡 训练技巧

1. **循序渐进**：从较低速度开始，逐步提高
2. **固定阅读时间**：每天坚持 15-30 分钟的训练
3. **多语言训练**：在中英文间切换，拓展阅读范围
4. **调整参数**：根据理解度调整行宽和行数
5. **舒适度优先**：确保眼睛舒适，避免疲劳

## 🐛 常见问题

### Q: PDF 文件无法打开？
A: 确保 PDF 文件包含可提取的文本内容，扫描版 PDF 不支持。

### Q: 速度设置对阅读效果的影响？
A: 
- 较低速度：有更多时间理解内容
- 较高速度：提高视觉敏锐度和反应速度

### Q: 如何选择视点移动方式？
A:
- 固定式：新手推荐，减少眼球疲劳
- 滚动式：高手推荐，更接近实际阅读习惯

### Q: 可以离线使用吗？
A: 需要网络连接以加载 PDF.js 库，建议下载到本地后离线使用（需要本地服务器）。
